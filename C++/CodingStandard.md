# C++ Coding Standard
## Rules
### 1. There will be a space before an if/for condition
#### Correct
```
if (condition)
{

}
```
```
for (condition)
{

}
```
```
while (condition)
{

}
```
```
do
{

} while (condition);
```
### 2. Empty brackets shall contain a single empty line between them
#### Correct
```
if (condition)
{

}
```
